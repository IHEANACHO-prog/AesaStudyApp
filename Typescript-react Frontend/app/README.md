# AESAStudy | Academic Discussion Platform

A high-performance React + TypeScript forum application built with Vite, powered by a Django backend.

## 🚀 Key Technologies
- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS + Shadcn/UI
- **Build Tool**: Vite (with HMR)
- **Icons**: Lucide React
- **Notifications**: Sonner

## 🛠 Project Configuration

### Path Aliases
We use the `@` alias for cleaner imports. 
`@/components/*` maps to `src/components/*`.
